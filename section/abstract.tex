\begin{abstract}
	\label{abstract}
	\texttt{hi-nginx}是一款基于\texttt{nginx}的通用服务器。
	
	它既是\texttt{ web server}，也是\texttt{application server}。
	
	它是\texttt{nginx}的超集，支持多种语言混合开发web应用：
	\begin{itemize}
		\item C++
		\item Python
		\item Lua
		\item Java
		\item PHP
		\item Javascript
	\end{itemize}
	
	它性能强劲，易于开发，部署方便。
	
	此文档已经很久没有跟随\texttt{hi-nginx}的新版本进行滚动更新，当前内容不一定是正确的。请前往\href{https://doc.hi-nginx.com/}{https://doc.hi-nginx.com/}查看最新说明。
\end{abstract}
